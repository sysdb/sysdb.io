sysdbd-facter(5) Manual Page

NAME

sysdbd-facter - query local facter fact values

SYNOPSIS

<LoadBackend "facter">
    Interval 3600
</LoadBackend>

DESCRIPTION

facter is a backend module which queries facter fact values from the local system. It periodically retrieves the values and submits them as host attributes.

CONFIGURATION

facter does not currently accept any configuration options.

sysdbd(1), sysdbd.conf(5), https://github.com/puppetlabs/cfacter
