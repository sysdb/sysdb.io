sysdbd-syslog(5) Manual Page

NAME

sysdbd-syslog - syslog plugin

SYNOPSIS

LoadPlugin syslog

DESCRIPTION

syslog is a plugin which connects to the local syslog daemon and forwards all log messages to it.

CONFIGURATION

syslog does not currently accept any configuration options.

SEE ALSO

sysdbd(1), sysdbd.conf(5)
